# RennAuktion

**Porsche auction intelligence platform.**

Live FMV estimates, BaT comp data, scatter graphs, urgency timers — purpose-built for serious Porsche buyers tracking active auctions across BaT, Cars & Bids, and pcarmarket.

## Stack

- **Frontend:** Static HTML/CSS/JS — generated by `auction_dashboard.py` in the [rennmarkt](https://github.com/OCX11/rennmarkt) repo
- **Hosting:** Cloudflare Pages
- **Auth (upcoming):** Supabase — shared auth layer with rennmarkt.net
- **Data:** Pushed from Mac Mini scraper every 2 min via git

## Deployment

Site deploys automatically on every push to `main` via Cloudflare Pages.

`index.html` is the primary output — generated by the scraper pipeline and committed by `git_push_dashboard.sh`.
